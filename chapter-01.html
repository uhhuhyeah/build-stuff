This is the part where we start to build some stuff.

The funny part is that we're going to start by learning how to destroy stuff.

How to delete a web page.

Locate the refresh button in Chrome. You'll be using it real soon.

Open up the Console. If you don't know what the Console is or how to open it, you should click Previous and do a quick review.

In the console, type in the following and before you hit the Enter key make sure you really know where the refresh button is located. It's the button you'll be hitting after you hit the Enter key.

{% highlight javascript %} document.write() {% endhighlight %}

Now hit the Enter key.

Explain what just happened.
